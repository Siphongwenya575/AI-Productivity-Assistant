# AI Productivity Assistant

## Project Overview

The AI Productivity Assistant is a web-based application that helps professionals automate common workplace tasks using Artificial Intelligence.

## Problem Statement

Professionals spend significant time drafting emails, summarizing meeting notes, planning tasks, and finding information. These repetitive activities reduce productivity and efficiency.

## Solution

This project provides an AI-powered assistant that automates workplace tasks and improves productivity.

## Features

### Smart Email Generator
Generates professional workplace emails based on user instructions.

### Meeting Notes Summarizer
Summarizes meeting notes into key points, decisions, and action items.

### Task Planner
Creates organized schedules and prioritizes tasks.

### AI Chatbot
Answers workplace-related questions and provides assistance.

## Technologies Used
